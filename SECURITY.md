# 安全策略

## 支持范围

项目尚未进入稳定版阶段。安全修复只会应用到 `main` 和最新发布版本，旧版用户应升级到最新 Release。

## 报告漏洞

请使用 GitHub 仓库的 **Security → Report a vulnerability** 私密报告功能。若该入口不可用，请通过维护者 GitHub 主页中的私密联系方式报告。

请提供受影响版本、复现条件、潜在影响和建议修复；不要提交真实密码、Cookie、访问令牌、学号或成绩数据。我们会尽快确认报告，并在修复发布前与你协调披露时间。

请勿为安全问题创建公开 Issue，也不要在未经许可的学校系统或他人账号上测试。
