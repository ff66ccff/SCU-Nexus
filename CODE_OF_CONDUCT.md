# 社区行为准则

我们致力于营造友好、尊重且不受骚扰的协作环境，不因年龄、身体条件、民族、性别认同、经验水平、国籍、外貌、种族、宗教、性取向或技术选择而区别对待参与者。

参与项目时，请保持建设性讨论，尊重不同观点，接受有根据的反馈，并把公共利益与用户隐私放在首位。侮辱、威胁、骚扰、公开他人隐私信息以及其他不专业行为不可接受。

维护者有权删除不当内容，并对屡次或严重违反本准则的参与者采取临时或永久限制。需要举报时，请通过维护者 GitHub 主页中提供的私密联系方式联系；不要在公开 Issue 中披露受害者身份或敏感证据。维护者会在保护相关人员隐私的前提下审查并处理举报。

本准则适用于项目仓库、Issue、Pull Request、讨论区，以及代表项目参与的其他公共空间。
